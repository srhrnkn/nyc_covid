This repository contains scripts in R and python that import case, hospitalization, and death counts from the [NYC Department of Health and Mental Hygiene's github site](https://github.com/nychealth/coronavirus-data) and visualize them year-over year.

[Key information about on the data sources and calculations is available here.](https://github.com/nychealth/coronavirus-data/blob/master/README.md#key-technical-notes)

As of October 15, 2025, the NYC Department of Health and Mental Hygiene is no longer updating the data in the coronavirus-data repository and instead is including it in the respiratory-illness-data repository.
